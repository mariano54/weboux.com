Lesson 1: Introduction

Let's get started on this epic journey.


Web Development is a very important skill to learn nowadays. Software companies are taking over the world, and new web services are being created every day. You never know when an opportunity can come up, and you need to launch a site in order to sell something.

My name is Mariano Sorgente, and I am a student in Stanford University. I have recently learned a lot of the basics of web development, especially from the CS184 class, and I wanted to pass on my skills, and explain everything in the easiest way possible. This guide will assume basic programming experience, but no web experience. It will take you all the way from a complete beginner, to being able to launch a flexible site on the internet.

We will go from buying a domain name, to designing a page, and coding the backend. I will teach you the basics of HTML, CSS, JavaScript, Node.js, GitHub, and Amazon Web Services. Don’t worry if you don’t know what any of these things are. You will very soon.
