Lesson 3: HTML

HyperText Markup Language.


Before we start, I recommend you download a text editor such as Sublime Text 2, Notepad++, or Texmate for Mac. Create a new file, and save it as index.html. Now we can begin.

HTML is sometimes referred to as a programing language, but it doesn’t even compare to the power of standard languages like C or Java. It is basically a way to organize all of the elements in a page, and put everything together. HTML consists of tags, which are created like this:

<tag> Content inside the tag </tag>

All tags start with <x> where x is the type of tag, and end with </x>. A tag contains content that is typically described by the tag name. We an have tags for a video, for a paragraph, for an image, and many other things. For example we will start creating our first html file like this:

<html>
    <body>
        Hello World
    </body>
</html>

Here you can see that there is an html tag, and then a body tag inside of that. Inside of the body tag, we write Hello World. The body tag will contain all of the content in a page. We can add some things such as links, or paragrags:

  • Link:

    <a href=\"http:/google.com\"> Click this to go to Google! </a>

  • Paragraph:

    <p> This is a paragraph </p>

As you can see, all these tags start, and then close with a /. Also, inside the opening <a> tag, which is for links, there is an href property. We set this equal to the source of where we want the link to point to. Most tags have attributes that you can change like this, in the opening tag.

We can add these things to our webpage like this:

<html>
    <body>
        <a href=”http:/google.com”> Click this to go to Google! </a>
        <a href=”http:/google.com”> Click this to go to Google! </a>
        <p> This is a paragraph. </p>
        <p> This is a paragraph. </p>
    </body>
</html>

As you can see, we have placed 2 links and 2 paragraphs in this page. The links will appear side by side, and then the two paragraphs will appear on their own line. This is because paragraphs have an automatic “enter” associated with them.

In order to test our html page, type this code into a file using a text editor, and save it as index.html. Open the file by double clicking it, which will open up a browser to view the file. We can test our html page like this, but as you can see in your browser, it is not online. It is displaying the local html file that we have created. In order to put it up online, we will have to create the backend/server (which we will do later on). Now let’s get to some more HTML Elements.
