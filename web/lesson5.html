Lesson 5: HTML Tags

HyperText Markup Language


Now we will examine the most used HTML elements, and how they are used. Here is a table of many html elements, these are specifically the ones that I have used the most, and this entire website is coded with elements in this table.


HTML element Function Common attributes
html Used to identify that there will be HTML content inside None
head Used to specify document metadata None
body Contains the content of the page None
img Displays an image on the screen src, width, height
div Container for anything id, class, width, height
span Container for anything, but inline (does not press enter) id, class
table Creates a table id, class, border
tr Table row, used withing table tags id, class, width, height
td Table cell, used within tr tags id, class, rows, cols, readonly
section Represents a section within a page id, class
meta Provides data about the HTML document name, content
link Relationship between document and an external source rel, type, href
script Defines a client side script src
canvas Creates a drawale canvas (well see this later when we learn JS) id, width, height
br Line break (like pressing enter) id, class, width, height,
textarea Used for a box for user input. id, class, width, height

You should take a look at all these tags, and look them up if you are curious. If you know these tags, you basically know HTML. In order for you to get a sense of how exactly all of these tags are used in practice, you should look at the source for this website. To do this, right click anywhere on the browser, and click on View Page Source. This will allow you to view all of the HTML code on the current page.

Now we will advance to learn about CSS. How to style HTML elements, and apply different looks to your website.

Before we move on, you should create a very basic HTML file so that you can style it in the next lesson. You should check out the source of the about page (which can be accessed in the top right), and copy and paste this into your own html file. Try to make sure you understand all of the HTML tags before we move on.
